## 🔧 Troubleshooting

### Issue: "Permission Denied" when running script
```bash
chmod +x extract_and_organize_complete.sh
```

### Issue: "unzip not found"
The script auto-installs, but manually:
```bash
sudo apt-get update && sudo apt-get install -y unzip p7zip-full unar
```

### Issue: Python3 not installed
```bash
python3 --version
# If not installed:
sudo apt-get install -y python3
```

### Issue: Disk space error
```bash
df -h                          # Check available space
du -sh *.zip *.7z *.rar        # Check archive sizes
du -sh ASSETS/                 # Check organized size
```

### Issue: RAR extraction fails
```bash
# Ensure unar is installed
sudo apt-get install -y unar
# Or use alternative:
which bsdtar  # Check for alternative
```

### Issue: Script stops mid-execution
```bash
# Check what failed
tail -100 extraction_manifest_*.log | grep -i error

# Rerun (it will continue from where it left off)
./extract_and_organize_complete.sh
```
