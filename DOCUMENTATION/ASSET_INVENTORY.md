# Asset Inventory

## Overview
Sprites and tilesets repository organized by type and theme. All assets are free/public domain resources for game development.

## Directory Structure

### ASSETS/tilesets/
Tileset collections for building levels and environments.

- **sci-fi-basic-battlefields/** - Basic sci-fi environment tiles
  - Contains ground, walls, and environmental elements
  - Recommended for: Top-down or isometric battlefield maps
  
- **ditharts-sci-fi-free/** - Ditharts Free Sci-Fi Tileset v01
  - Free sci-fi themed tileset by Ditharts
  - High quality pixel art
  
- **cyberpunk/** - Cyberpunk themed tiles
  - Neon and industrial aesthetic
  - Suitable for: Dystopian cyberpunk environments

### ASSETS/sprites/
Individual sprites and sprite sheets for characters, enemies, and interactive elements.

- **characters/** - Character and NPC sprites
- **environment/** - Environmental decoration sprites (trees, rocks, furniture, etc.)
- **props/** - Interactive objects and items

### ASSETS/interiors/
Interior decoration and room assets.

- **Inside_C.png** - Sci-Fi interior tileset (~70 different interior tiles)
  - Includes: Beds, desks, walls, doors, storage units, floor tiles
  - Perfect for: Spaceship interiors, sci-fi buildings

### ASSETS/ui/
User interface elements (buttons, menus, HUD elements)

## Original Archives
All original archive files are preserved in ARCHIVES/ for backup and reference:
- ARCHIVES/original-zips/ - Contains all .zip files
- ARCHIVES/original-7z/ - Contains all .7z files

## Recommended Usage

### For Top-Down RPG:
Use sci-fi-basic-battlefields tilesets + cyberpunk tilesets

### For Spaceship Game:
Use interiors (Inside_C.png) + ditharts sci-fi tileset

### For General Sci-Fi:
Mix assets from all sources as needed
